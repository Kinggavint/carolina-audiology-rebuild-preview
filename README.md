# Carolina Audiology Associates

Proof website for Carolina Audiology Associates — an independent audiology practice in Rocky Mount, NC, led by Dr. Melissa Palmer, Au.D.

## Live preview
https://kinggavint.github.io/carolina-audiology-proof/

## Pages
- Home — overview, services snapshot, why-us, FAQs
- Services — hearing evaluation, tinnitus, hearing aids, earwax removal, hearing protection
- About — Dr. Melissa Palmer and practice values
- Hearing Health — patient education on signs of hearing loss, cognition link, hearing aid basics
- Contact — appointment request form + hours + insurance accepted

## Stack
Static HTML + CSS. No frameworks, no build step. Hosted on GitHub Pages.
